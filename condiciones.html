Condiciones del Servicio

UrbanSafe

Bienvenido a UrbanSafe. Al utilizar nuestra aplicación o sitio web, aceptas cumplir con las siguientes condiciones del servicio. Te recomendamos leerlas cuidadosamente.

1. Uso de la aplicación

UrbanSafe es una aplicación diseñada para permitir a los usuarios reportar situaciones de riesgo y mejorar la seguridad urbana mediante el uso de tecnología y geolocalización.

El usuario se compromete a utilizar la aplicación únicamente con fines legítimos y relacionados con la seguridad ciudadana.

2. Responsabilidad del usuario

  • No realizar reportes falsos o malintencionados.
  • No utilizar la aplicación para actividades ilegales.
  • Proporcionar información veraz cuando se realicen reportes.

3. Limitación de responsabilidad

UrbanSafe funciona como una herramienta de apoyo para reportar situaciones de riesgo. No garantizamos una respuesta inmediata por parte de las autoridades ni nos hacemos responsables por las acciones de terceros.

4. Uso de la información

La información recopilada por la aplicación se utilizará únicamente para mejorar el funcionamiento del servicio y facilitar la comunicación de incidentes de seguridad.

5. Modificaciones del servicio

UrbanSafe se reserva el derecho de modificar o suspender el servicio en cualquier momento sin previo aviso.

6. Cambios en las condiciones

Estas condiciones del servicio pueden actualizarse en cualquier momento. Los cambios serán publicados en esta página.
